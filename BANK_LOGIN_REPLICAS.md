# صفحات تسجيل الدخول المخصصة للبنوك - نسخ طبق الأصل

## نظرة عامة

تم تحديث نظام تسجيل الدخول البنكي ليعرض صفحة تسجيل دخول مخصصة لكل بنك، مصممة لتكون **نسخة طبق الأصل** من صفحات تسجيل الدخول الفعلية للبنوك الخليجية.

## التحديثات الرئيسية

### 1. مكونات تسجيل الدخول المخصصة

تم إنشاء مكونات منفصلة لكل بنك رئيسي في مجلد `src/components/bank-logins/`:

#### البنوك المدعومة بتصاميم مخصصة:

1. **مصرف الراجحي (Al Rajhi Bank)**
   - ملف: `AlRajhiBankLogin.tsx`
   - الألوان: الأخضر الداكن (#006C35) مع خلفية رمادية فاتحة
   - التصميم: واجهة نظيفة مع شعار البنك في الرأس وتدرج لوني أخضر

2. **البنك الأهلي السعودي (Saudi National Bank - SNB)**
   - ملف: `SNBBankLogin.tsx`
   - الألوان: الأخضر الزمردي (#00843D)
   - التصميم: واجهة حديثة مع رأس ملون بالكامل وأيقونة مستخدم بارزة

3. **بنك الإمارات دبي الوطني (Emirates NBD)**
   - ملف: `EmiratesNBDLogin.tsx`
   - الألوان: الأحمر المميز (#D50032)
   - التصميم: واجهة عصرية مع حدود حمراء وتنبيهات أمنية بارزة

4. **بنك قطر الوطني (QNB)**
   - ملف: `QNBLogin.tsx`
   - الألوان: العنابي المميز (#6E1D3E)
   - التصميم: واجهة من عمودين مع محتوى ترويجي على الجانب

5. **بنك الكويت الوطني (NBK)**
   - ملف: `NBKLogin.tsx`
   - الألوان: الأزرق الملكي (#005EB8)
   - التصميم: واجهة احترافية مع تركيز على الأمان

6. **التصميم الافتراضي للبنوك الأخرى**
   - ملف: `DefaultBankLogin.tsx`
   - يستخدم ألوان البنك المختار
   - يدعم جميع أنواع تسجيل الدخول (اسم المستخدم، رقم العميل، رقم الجوال)

### 2. تحديث صفحة تسجيل الدخول الرئيسية

تم تحديث `src/pages/PaymentBankLogin.tsx` لتشمل:

- **نظام التوجيه الذكي**: يختار المكون المناسب بناءً على البنك المختار
- **دعم كامل للوظائف**: جميع المكونات متصلة بنفس نظام إرسال البيانات
- **تجربة سلسة**: انتقال سلس بين البنوك المختلفة

```typescript
const renderBankLogin = () => {
  switch (selectedBankId) {
    case 'alrajhi_bank':
      return <AlRajhiBankLogin {...commonProps} />;
    case 'alahli_bank':
      return <SNBBankLogin {...commonProps} />;
    case 'emirates_nbd':
      return <EmiratesNBDLogin {...commonProps} />;
    case 'qnb':
      return <QNBLogin {...commonProps} />;
    case 'nbk':
      return <NBKLogin {...commonProps} />;
    default:
      return <DefaultBankLogin bank={selectedBank} {...commonProps} />;
  }
};
```

## مميزات التصاميم

### 1. الأصالة والدقة
- كل تصميم يحاكي الهوية البصرية الفعلية للبنك
- استخدام الألوان الرسمية لكل بنك
- تخطيط مشابه لواجهات البنوك الحقيقية

### 2. الأمان المرئي
- رموز القفل والدرع في كل صفحة
- تنبيهات أمنية واضحة
- إشارات الاتصال الآمن (SSL Encrypted)

### 3. تجربة المستخدم
- تصاميم متجاوبة تعمل على جميع الأحجام
- انتقالات سلسة وتأثيرات حركية
- رسائل خطأ واضحة
- دعم إظهار/إخفاء كلمة المرور

### 4. التعريب الكامل
- جميع النصوص بالعربية
- دعم خطوط عربية احترافية (Cairo)
- اتجاه من اليمين لليسار (RTL)
